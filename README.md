# Netflix (Create React App + Tailwind CSS)

This project is a React app scaffolded in a Create React App style and configured with Tailwind CSS.

## Run locally

```bash
cd netflix
npm install
npm start
```

## Tailwind setup

- Tailwind directives are in `src/index.css`.
- Tailwind content paths are configured in `tailwind.config.js`.
- PostCSS plugins are configured in `postcss.config.js`.

## Example UI

`src/App.js` includes a Netflix-style hero and trending cards using Tailwind utility classes.
